O projeto Streaming de Música é um sistema desenvolvido em Java com foco em Programação Orientada a Objetos (POO), simulando o funcionamento básico de uma plataforma de reprodução musical, semelhante aos serviços de streaming atuais.

O sistema permite o cadastro e gerenciamento de músicas e usuários, diferenciando contas do tipo Free e Premium. Usuários Free podem reproduzir músicas com limitações e exibição de anúncios, enquanto usuários Premium possuem acesso a benefícios como reprodução sem anúncios e possibilidade de download de músicas.

Além disso, o projeto utiliza conceitos fundamentais de POO como criação de classes personalizadas, atributos e métodos de instância, construtores com uso da palavra-chave this, herança, polimorfismo, encapsulamento e manipulação de ArrayList para armazenamento de objetos.

A aplicação foi estruturada para representar de forma prática situações do mundo real, promovendo uma melhor compreensão da modelagem orientada a objetos e das regras de negócio envolvidas em um sistema de streaming musical.
